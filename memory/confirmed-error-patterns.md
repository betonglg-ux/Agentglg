# Confirmed Error Patterns

## Статус на 2026-04-29

Пока в памяти не зафиксированы подтвержденные пользователем типовые ошибки именно по проверке PDF-протоколов.

Правило заполнения:
- сюда добавлять только те паттерны ошибок, которые пользователь явно подтвердил как реальные;
- не переносить сюда предположения, спорные случаи и сырые догадки;
- если подтвержденный паттерн относится к конкретному типу протокола, обязательно указывать этот тип.
