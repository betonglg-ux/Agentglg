# Экспорт заметок по шаблонам

Назначение: сюда нужно переносить устойчивые особенности Excel-шаблонов и правил сверки PDF против шаблонов, если эти знания важны для будущих похожих агентов.

## Обновлено: 2026-05-05

### УК / УИ / бетон

- Для УК-протоколов по схеме Г с приложением Б нужно проверять, что значения `Hi` из таблицы 1 приложения Б указаны в отчете(ах), а данные таблицы 2 приложения Б идентичны данным отчета.
- Для УК-протоколов с приложениями-схемами принадлежность `Приложения А` к конкретному отчету нужно отслеживать по номеру в нижнем колонтитуле.
- Если у УК-протокола только одно приложение со схемой, некорректное окончание `Схема` / `Схемы` нужно фиксировать именно в приложении.
- Для УК/УИ-протоколов, если дата бетонирования указана диапазоном, возраст указывается по крайней дате бетонирования.
- Для шаблона `2.0 ИПС универсальная` в УИ-протоколах запись диапазонов вида `27,3 - 30,6 МПа` и `71,1 - 79,8%`, где единица измерения указана только после второго значения диапазона, является допустимой и не считается ошибкой.
- Для УК-протоколов с `Приложением А` нужно отдельно сверять все даты на схеме с датами бетонирования в основном протоколе.

### КУ

- Для КУ-протоколов вывод обязан соответствовать наименованию участка в таблице.
- Если в таблице указан один участок, форма `обследуемых участков` в выводе является ошибкой; множественное число допустимо только при нескольких участках.
- В выводе обязательно должен быть указан тип участка из таблицы.
- Для обратной засыпки корректная логика формулировки: `...на обследованном участке обратной засыпки...`.
- Для КУ-протоколов нельзя придумывать названия вывода от себя. Нужно использовать шаблонную фразу, где меняется только число и тип основания.
- Подтвержденные пары:
  - `одно основание / песчаное основание -> Коэффициент уплотнения обследуемого участка песчаного основания, на момент...`
  - `несколько оснований / песчаное основание -> Коэффициент уплотнения обследуемых участков песчаного основания, на момент...`
  - `одно основание / обратная засыпка -> Коэффициент уплотнения обследуемого участка обратной засыпки, на момент...`
  - `несколько оснований / обратная засыпка -> Коэффициент уплотнения обследуемых участков обратной засыпки, на момент...`
  - `одно основание / грунтовое основание -> Коэффициент уплотнения обследуемого участка грунтового основания, на момент...`
  - `несколько оснований / грунтовое основание -> Коэффициент уплотнения обследуемых участков грунтового основания, на момент...`

### ВС

- Для ВС-протоколов отсутствие поля `Материал по проекту` не считать ошибкой. Если материал известен и его логично указать, добавлять только напоминание в конце отчета и не выносить этот случай в список замечаний по страницам.
- Для шапки ВС-протоколов не проверять упоминание ГОСТ в шапке и не выносить по этой строке замечание.

### Общие шаблонные и оформительские правила

- Номер отчета или протокола должен иметь вид `номер.подномер.идентификатор`, например `253.1.УК`. Подномер начинается с `1` и дальше идет по количеству протоколов в серии.
- Для протоколов типа `№.КП` нижний колонтитул не предусмотрен.
- По отметкам: абсолютные отметки писать без знака, в метрах, с точностью до тысячных, например `140,000`. Относительные отметки писать со знаком `+` или `-`, кроме `0,000`.
- Разделитель в отметках использовать только запятую: запись `140,200` корректна, запись `140.200` некорректна.
- Для их протоколов допустимо, когда у конструкции возраст `28` суток, а в выводе написано `более 28 суток`.
- Формулировки в выводах `менее 28 суток` и `более 28 суток` не считать ошибкой.
- Служебную нумерацию страниц в нижнем колонтитуле не считать ошибкой.
