# Экспорт подтверждённых паттернов ошибок

Назначение: сюда нужно переносить из Memory подтверждённые пользователем типовые ошибки, которые агент должен особенно внимательно проверять в будущих запусках и в будущих похожих агентах.

## Статус
Пока стартовая заготовка. Заполняется по мере накопления подтверждённых паттернов.
