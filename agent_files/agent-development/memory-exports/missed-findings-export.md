# Экспорт журнала пропущенных замечаний

Назначение: сюда нужно переносить из Memory реальные случаи, когда агент пропустил очевидные ошибки, чтобы эти случаи можно было использовать при восстановлении и дообучении похожих агентов.

## Статус
Пока стартовая заготовка. Заполняется по мере накопления подтверждённых пропусков.
