# Экспорт Memory для GitHub-зеркала

Назначение: эта папка хранит переносимые выгрузки из Memory, которые важны для восстановления или клонирования похожего агента.

## Что складывать сюда
- подтверждённые паттерны ошибок;
- журнал пропущенных замечаний;
- заметки по шаблонам;
- подтверждённые поправки пользователей.

## Принцип
В Memory агент продолжает работать как обычно. В эту папку нужно выгружать только устойчивые и полезные для будущих агентов данные.

## Минимальный состав файлов
- `confirmed-error-patterns-export.md`
- `missed-findings-export.md`
- `template-notes-export.md`
- `user-corrections-export.md`
