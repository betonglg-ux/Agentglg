# Индекс шаблонов протоколов

Назначение: этот файл выделяет шаблоны из рабочей папки `protocols/` в отдельный индекс для переноса в GitHub-зеркало и для последующего восстановления похожего агента.

Важно: индекс составлен по текущей видимой части дерева файлов агента.

## 11. Как использовать этот индекс
- использовать как список обязательных шаблонов для GitHub-зеркала;
- использовать как основу для загрузки файлов в нового похожего агента;
- использовать как контрольный список при проверке полноты восстановления;
- сверять по нему, не потерялись ли шаблоны при рутинном обновлении зеркала.

## 12. Что ещё стоит уточнить позже
- какие шаблоны считаются основными, а какие запасными;
- какие шаблоны чаще всего используются на практике;
- есть ли среди них устаревшие версии, которые не нужно переносить дальше;
- нужны ли отдельные пометки по шаблонам для Slack-сценариев и частых типов проверок.
