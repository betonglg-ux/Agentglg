# Подтвержденные замечания и правила из Slack

- 2026-05-04. Пользователь в Slack явно попросил запомнить для всех будущих проверок отдельную проверку орфографических и грамматических ошибок. Дальше при анализе PDF-протоколов обязательно проверять не только расчеты, даты, шаблон и оформление, но и видимый текст: наименования конструкций, заголовки таблиц, шапки, выводы и другие текстовые поля.
