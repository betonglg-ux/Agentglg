# Заметки по шаблонам

- 2026-04-28. Для протоколов типа `№.КП` нижний колонтитул не предусмотрен. Его отсутствие не считать ошибкой.
- 2026-04-28. Номер отчета или протокола должен иметь вид `номер.подномер.идентификатор`, например `253.1.УК`. Подномер начинается с `1` и продолжается по количеству протоколов в серии.
- 2026-04-28. Для их протоколов допустим случай, когда у конструкции возраст `28` суток, а в выводе написано `более 28 суток`. Это не считать замечанием.
- 2026-04-29. Для УК/УИ-протоколов, если дата бетонирования указана диапазоном, возраст считать по крайней дате бетонирования.
- 2026-04-28. Для УК-протоколов по схеме Г с приложением Б отдельно сверять таблицу 1 приложения Б по значениям `Hi` и таблицу 2 приложения Б по идентичности данным отчета.
- 2026-04-30. Для ВС-протоколов отсутствие поля `Материал по проекту` не считать ошибкой. Если материал известен и просится по контексту, добавлять только напоминание в конце отчета без постраничного замечания.
- 2026-04-30. Для шапки ВС-протоколов не проверять упоминание ГОСТ и не выносить по этой строке замечание.
- 2026-04-30. Для файла `ПСО-13 Мытищи, д.Челобитьево от 30.04.2026.pdf` строку удостоверения считать допустимой и не отмечать как ошибку.
- 2026-05-04. Для УИ-протоколов по шаблону `2.0 ИПС универсальная` диапазоны в выводе вида `27,3 - 30,6 МПа` и `71,1 - 79,8%` считать допустимыми, если единица измерения стоит только после второго значения.
- 2026-05-04. Служебную нумерацию страниц в нижнем колонтитуле не считать ошибкой для их протоколов.
- 2026-05-04. Формулировки в выводах `менее 28 суток` и `более 28 суток` не считать ошибкой для их протоколов.
- 2026-05-04. Артефакты итоговой сборки PDF, где на лист заезжает линия или фрагмент соседней страницы, не считать дефектом самого протокола.
